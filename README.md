# Tomato Runner

Authors: Jake Ruiz, Dana Shakrovsky


### Project Description
Unity source code for the platform game Tomato Runner, available on itch.io: https://daysky.itch.io/tomatorunner 

Tomato Runner is a parkour-style platformer game about a tomato who fell out of a farmer's truck, where the player must navigate through the obstacles to pass 
each level and help Mr. Tomato get back home to his family!
